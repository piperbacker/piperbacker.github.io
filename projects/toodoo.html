Toodoo Chrome Extension

Build to-do lists with a colorful interface inspired by the day night cycle built using Typescript and React.

Background

The company blog page aims to provide potential customers and developers with the latest news on software releases, company events, and most importantly to highlight competitive benchmarks and showcase the results of projects undertaken with clients in a multitude of industries.
This project was accomplished in my current position as a frontend software developer at Hexaly (ex-Localsolver).

Tools

• React
• Typescript
• Figma
• Affinity Designer
• Material Icons

Design

I wanted a sleek and aesthetically pleasing way to maintain my to-do list through Chrome.

1: Easily filter articles by category, industry, and problem type with a clean and intuitive sidebar design. Sort articles by post date or by alphabetical order.

2: Implement a 2 column grid to display more articles at once, with more compact and straightforward cards.

3: Maximize the space in the sidebar by adding brand social media accounts. Include a call to action to encourage the user to start a free trial

check out the live blog page here

Results

The refreshed blog landing page encourages the user to interact and seek out the articles that will interest them with an intuitive filtering interface. The new two-column layout displays more articles at once providing a more polished and concise look and feel.
