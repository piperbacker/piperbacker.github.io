Hexaly Blog Refresh

Redesign the Hexaly blog page to provide an intuitive and adapted user experience while maintaining the existing visual identity.

Background

The company blog page aims to provide potential customers and developers with the latest news on software releases, company events, and most importantly to highlight competitive benchmarks and showcase the results of projects undertaken with other clients in similar industries.
This was a project undertaken as part of my current position as a frontend software developer at Hexaly (ex-Localsolver).

The Goal

When a user arrives on the blog page, the user should be able to quickly and efficiently identify blog articles that are relevant to them. The blog page should take full advantage of the space available and include direct calls to action to encourage the user to engage with the company brand.

The initial blog page layout

The Starting Point

The initial blog page includes a simple, single column layout which displays the articles in order of the date they were posted. There are no tools provided to refine a search. The article cards do not make efficient use of the available space and including the beginning paragraph of the article is unnecessary given the detailed article titles.

Figma mockup with solutions incorporated

The Solution

When a user arrives on the blog page, the user should be able to quickly and efficiently identify blog articles that are relevant to them. The blog page should take full advantage of the space available and include direct calls to action to encourage the user to engage with the company brand.

Solution 1: Easily filter articles by type, industry, and problem type with a clean and intuitive sidebar design. Sort articles by post date or by alphabetical order.

Solution 2: Implement a 2 column grid to display more articles at once, with more compact and straightforward cards.

Solution 3: Maximize the space in the sidebar by adding brand social media accounts Include a call to action to encourage the user to start a free trial

The Results

The blog page encourages the user to interact and seek out the articles that will interest them by providing an intuitive filtering interface and displaying more articles at once in a clean and straightforward manner.
